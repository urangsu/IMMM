# Frame Design QA

## Visual Quality
- 프레임이 기능 테스트용 도형처럼 보이지 않는가
- 실제 네컷사진처럼 여백이 안정적인가
- 사진 영역이 충분히 큰가
- 장식이 사진을 과하게 가리지 않는가
- 로고/워터마크가 과하지 않은가
- 모바일 썸네일에서도 프레임 분위기가 보이는가

## Layout
- 1x4
- 2x2
- 1x3
- 1x1

## Render Parity
- Setup
- Store
- Designer
- Deco
- Result
- Save Image

## Fail Criteria
- 배경이 두 번 그려져 진해짐
- 워터마크가 두 번 나옴
- 사진 슬롯 위치가 화면마다 다름
- 프레임 선택이 레이아웃 변경 후 사라짐
- 저장한 My Frame이 다시 불러오면 달라짐
