# IMMM 포토부스 개발 태스크 리스트

## 🔴 최우선 해결 과제 (P0)
- [x] **드로잉 모드 크래시 수정**: `renderStroke` 스코프 문제로 인한 화면 멈춤 및 하얀 화면 버그 수정 완료.
- [x] **셀렉 화면 필터 미적용 수정**: WebGL 엔진에서 직접 스냅샷을 찍도록 개선하여 필터가 누락되는 문제 해결.
- [x] **SNS 공유 아이콘 스타일 수정**: 인스타그램/카톡 아이콘을 검은색 라인 스타일로 변경.

## 🟠 주요 기능 개선 (P1)
- [ ] **배경 블러 (Bokeh)**: MediaPipe Selfie Segmentation을 통합하여 인물 제외 배경 블러 효과 추가.
- [ ] **촬영 리빌 애니메이션**: 사진 촬영 후 폴라로이드가 흔들리며 나타나는 애니메이션 및 폭죽 효과 추가.
- [ ] **QR 코드 커스텀**: 결과 화면 QR 코드 중앙에 썸네일 이미지 삽입.

## 🟡 사용성 및 품질 (P2)
- [ ] **영상 저장 로직 점검**: 실제 비디오 파일로 인코딩되어 다운로드되는지 최종 확인.
- [ ] **BGM 추가**: 비디오 녹화 시 배경 음악 합성 기능.
- [ ] **안면 인식 보정 정밀화**: 홍조(Blush) 위치가 얼굴 움직임에 더 부드럽게 따라오도록 보간 로직 추가.

## ✅ 완료된 항목
- [x] WebGL 기반 실시간 고사양 필터 (BlurCAM, Twilight 등) 구현.
- [x] 삼성 인터넷 비디오 렌더링 버그 수정.
- [x] 스티커 상태 유지 및 촬영 화면 오버레이 매핑.
- [x] 드로잉 성능 최적화 (Sparkle 브러시).
