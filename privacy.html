개인정보처리방침 (Privacy Policy)

시행일자: 2026년 5월 14일

1. 서비스 개요

IMMM(이하 "서비스")은 웹 기반의 네컷 포토부스 서비스입니다.

2. 카메라 권한 사용 목적

서비스는 포토부스 촬영 기능을 제공하기 위해 사용자의 카메라 권한을 요청합니다. 카메라는 브라우저에서 실시간 프리뷰를 제공하고 사진을 캡처하는 용도로만 사용됩니다.

3. 사진 및 데이터 처리 (On-Device)

  • 모든 사진 촬영, 필터 적용, 얼굴 인식(스티커 기능)은 사용자의 브라우저/기기 내에서만 처리(On-Device)됩니다.
  • 서비스는 사용자의 얼굴 데이터나 사진 원본을 외부 서버로 전송, 저장, 또는 수집하지 않습니다.

4. 서버 업로드 및 클라우드 공유

현재 서비스는 서버 업로드, QR 코드 공유, 클라우드 저장 기능을 제공하지 않습니다. 모든 결과물은 사용자의 기기에만 존재합니다.

5. 사진 저장

사용자가 최종 결과 화면에서 명시적으로 '저장' 버튼을 눌렀을 때만 기기(갤러리/다운로드 폴더)에 파일로 저장됩니다.

6. Analytics (통계 수집)

서비스 개선을 위해 앱 실행, 촬영 완료 등의 익명화된 이벤트 로그를 수집할 수 있습니다. 단, 이 과정에서 개인 식별 정보(PII), 기기 정보, 사진, 얼굴 데이터 등은 절대 수집되지 않습니다.
