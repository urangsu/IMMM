# Field QA Script

Use this on a real device or browser profile.

## Frame samples

1. Open the Frame Store.
2. Check that `Featured`, `Free`, `My Frames`, and `Premium` are accessible.
3. Open the sample page and compare the rendered geometry with the live UI.

## Designer

1. Enter Frame Designer.
2. Change background, slot geometry, decorations, text, and watermark.
3. Save to My Frames.
4. Reload the saved frame.
5. Duplicate and edit it.
6. Soft delete it and confirm it disappears from the default list.

## Pack import/export

1. Export a pack JSON from My Frames.
2. Re-import the JSON.
3. Confirm invalid payloads with photo data URLs are rejected.

## Result path

1. Capture with a selected frame.
2. Confirm Deco and Result previews match the designer preview.
3. Save and share the result.
